# 症状① 招待メール送信

```
new row violates row-level security policy for table "invite_codes"
```

RLSをOFFにすると正常にメール送信できる。

つまり

- invite_codes INSERT自体は正常
- RLSで拒否されている

ことが判明。

# 最初に疑ったこと

## ① invite_codes RLS

RLS

```
INSERT
WITH CHECK is_admin()
```

is_admin()

```sql
select exists (
    select 1
    from users
    where id = auth.uid()
      and role='admin'
)
```

確認結果

```
auth.uid()
```

が

```
NULL
```

になっていた。
